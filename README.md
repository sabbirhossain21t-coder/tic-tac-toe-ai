# 🎮 Tic Tac Toe AI (Pro Version)

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**, featuring an AI opponent powered by the **Minimax algorithm**.

---

## 🚀 Features

- 🤖 Play against AI (Smart logic)
- 📱 Fully responsive design (mobile + desktop)
- 🏆 Scoreboard (You vs AI)
- 🔄 Restart game anytime
- ⚡ Clean and smooth UI
- 🧠 Minimax-based AI

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
